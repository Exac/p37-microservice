# Microservice

## Scripts

#### Install & generate config typings
```bash
$ npm install
```

#### Run unit tests
```bash
# Run once
$ npm run test

# Watch files for changes
$ npm run test:watch
```

#### Start development server
```bash
$ npm run start

# With debug
$ npm run start:debug
```

#### Lint & compile
```bash
$ npm run lint
```

#### Run in production
```bash
$ node dist
```
